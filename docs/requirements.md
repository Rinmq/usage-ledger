# 要件メモ

## 概要

UsageLedger は、架空SaaSのAPI利用量を日別に記録し、月次で請求金額を計算する業務システムを想定したポートフォリオです。

## 主な機能

- 顧客管理
- サービス管理
- プラン管理
- 契約管理
- 日別利用量管理
- 月次利用量集計
- 請求金額計算
- 契約期間外利用の不整合検知

## 業務ルール

- 顧客は複数のサービスを契約できる
- サービスごとに複数の料金プランを持てる
- 契約には開始日と終了日を持たせる
- 利用量は顧客・サービス・利用日単位で記録する
- 月次請求では対象月の利用量を集計する
- 無料枠を超えた利用量に対して従量課金する
- 同一顧客・同一請求月の請求は重複作成しない
- 契約期間外の利用量は請求対象にせず、不整合データとして検知する

## 対象外

現時点では、認証、画面、決済連携、PDF請求書出力、メール送信は対象外です。
